# 🧳 AI Trip Planner ✈️

**AI Trip Planner** is your smart travel assistant. Just describe your ideal trip, and our AI builds a personalized travel itinerary using live data — destinations, weather, hotels, activities, and more. Save hours of planning with one smart prompt!

---

## 🔥 Features

- 🗺️ Natural Language Input: Just tell the AI where you want to go and what you like.
- 🤖 Personalized Itinerary: Fully AI-generated, day-by-day travel plans.
- 🌦️ Live Weather Info: Real-time weather data to adjust plans.
- 🏨 Hotel & Restaurant Suggestions: Based on location, rating, and budget.
- 📍 Interactive Map View: See your route and stops visually.
- 📆 Save & Share: Save your trip and share it with friends or print it.

---

## 🛠️ Tech Stack

Here’s a tech stack that’s both beginner-friendly and powerful:

### 💡 Frontend (User Interface)
- **React.js** – Modern, component-based UI framework
- **Tailwind CSS** – Fast and beautiful styling
- **React Router** – Page routing
- **Axios** – For API calls

### ⚙️ Backend (API & AI)
- **Node.js + Express** – Lightweight and fast server
- **OpenAI GPT-4 API** – Generates travel plans from natural language
- **Weather API (e.g., OpenWeatherMap)** – Live weather data
- **Travel APIs (optional)**:
  - [Amadeus](https://developers.amadeus.com/) for flights, hotels, etc.
  - [TripAdvisor API (via RapidAPI)](https://rapidapi.com/) for places and activities
  - [Google Places API](https://developers.google.com/maps/documentation/places/web-service/overview)

### 🗃️ Database (optional for storing user trips)
- **MongoDB** – Flexible NoSQL database
- OR
- **Firebase Firestore** – Easy integration for beginners
